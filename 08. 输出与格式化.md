# Go语言的输出与格式化占位符

## fmt中的几种print
1. Print:   输出到控制台,不接受任何格式化操作
2. Println: 输出到控制台并换行
3. Printf : 只可以打印出格式化的字符串。只可以直接输出字符串类型的变量（不可以输出别的类型）
4. Sprintf：格式化并返回一个字符串而不带任何输出
5. Fprintf：来格式化并输出到 io.Writers 而不是 os.Stdout

## 通用的占位符
+ %v    值的默认格式。
  + 默认格式%v下，对于不同的数据类型，底层会去调用默认的格式化方式：
  
    | 类型                    | 占位符                     |
    | ----------------------- | -------------------------- |
    | bool                    | %t                         |
    | int, int8 etc.          | %d                         |
    | uint, uint8 etc.        | %d, %x if printed with %#v |
    | float32, complex64, etc | %g                         |
    | string                  | %s                         |
    | chan                    | %p                         |
    | pointer                 | %p                         |
  + 如果是复杂对象的话,按照如下规则进行打印：
    | 类型             | 输出                         |
    | ---------------- | ---------------------------- |
    | struct           | {field0 field1 ...}          |
    | array, slice     | [elem0 elem1 ...]            |
    | maps             | map[key1:value1 key2:value2] |
    | pointer to above | &{}, &[], &map[]             |
 
+ %+v   类似%v，但输出结构体时会添加字段名
+ %#v　 类型+值对象
+ %T    相应值的类型的Go语法表示 
+ %%    百分号,字面上的%,非占位符含义
  
## 常用类型
+ 整数类型
    | 占位符 | 意义                                       |
    | ------ | ------------------------------------------ |
    | %b     | 二进制表示                                 |
    | %c     | 相应Unicode码点所表示的字符                |
    | %d     | 十进制表示                                 |
    | %o     | 八进制表示                                 |
    | %q     | 单引号围绕的字符字面值，由Go语法安全地转义 |
    | %x     | 十六进制表示，字母形式为小写 a-f           |
    | %X     | 十六进制表示，字母形式为大写 A-F           |
    | %U     | Unicode格式：123，等同于 "U+007B"          |
+ 浮点数
    | 占位符 | 意义                                                  |
    | ------ | ----------------------------------------------------- |
    | %b     | 无小数部分、二进制指数的科学计数法，如-123456p-78     |
    | %e     | 科学计数法，例如 -1234.456e+78                        |
    | %E     | 科学计数法，例如 -1234.456E+78                        |
    | %f     | 有小数点而无指数，例如 123.456                        |
    | %g     | 根据情况选择 %e 或 %f 以产生更紧凑的（无末尾的0）输出 |
    | %G     | 根据情况选择 %E 或 %f 以产生更紧凑的（无末尾的0）输出 |
+ 布尔型
    | 占位符 | 意义          |
    | ------ | ------------- |
    | %t     | true 或 false |
+ 字符串
    | 占位符 | 意义                                   |
    | ------ | -------------------------------------- |
    | %s     | 字符串或切片的无解译字节               |
    | %q     | 双引号围绕的字符串，由Go语法安全地转义 |
    | %x     | 十六进制，小写字母，每字节两个字符     |
    | %X     | 十六进制，大写字母，每字节两个字符     |
+ 指针
    | 占位符 | 意义                  |
    | ------ | --------------------- |
    | %p     | 十六进制表示，前缀 0x |
+ 其他标志
    | 占位符 | 意义                                                       |
    | ------ | ---------------------------------------------------------- |
    | +      | 总打印数值的正负号；对于%q（%+q）保证只输出ASCII编码的字符 |
    | -      | 左对齐                                                     |
    | #      | 备用格式：见下注释                                         |
    | ' '    | 空格：见下注释                                             |
    | 0      | 填充前导的0而非空格；对于数字，这会将填充移到正负号之后    |
  + 占位符`#`：为八进制添加前导 0（%#o），为十六进制添加前导 0x（%#x）或0X（%#X），为 %p（%#p）去掉前导 0x；对于 %q，若 strconv.CanBackquote 返回 true，就会打印原始（即反引号围绕的）字符串；如果是可打印字符，%U（%#U）会写出该字符的Unicode编码形式（如字符 x 会被打印成 U+0078 'x'）。
  + 占位符` `: (空格）为数值中省略的正负号留出空白（% d）；以十六进制（% x, % X）打印字符串或切片时，在字节之间用空格隔开
